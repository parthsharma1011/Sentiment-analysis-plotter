## Getting Started

Clone this repository in your system, run the sentiment_analyser.py first to get the idea of how the program 
works, in the later stages run the tweetest.py, in tweetest.py provide the API keys

### Prerequisites

```
Python3.3 & >
Pandas
Sklearn
nltk
ray 
matplotlib
tweepy
```
